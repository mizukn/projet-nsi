# Projet de NSI
Clone de agar.io, realise en TypeScript avec ExpressJS et Socket.io

# Comment lancer le projet
1. Cloner le projet
```bash
git clone
```

2. Installer les dependances
```bash
npm install
```

3. Build le projet
```bash
tsc
```

4. Lancer le jeu
```bash
node dist/server.js
```

5. Ouvrir le navigateur
```bash
http://localhost:3000
```
